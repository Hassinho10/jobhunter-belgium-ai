# CV Template

Ce dossier accueillera le CV HTML/CSS de base.

## Fichiers

- `index.html` : CV HTML de base utilisé comme modèle principal.
- `CV_Hassan_Chajai_BASE.html` : copie de référence du CV fourni.
- `style.css` : fichier conservé pour une éventuelle séparation future des styles.
- `data-example.json` : exemple de données structurées.

Le CV fourni contient déjà son style dans le fichier HTML. Une séparation HTML/CSS pourra être faite plus tard si elle devient utile pour l'automatisation.

## Règle importante

Les adaptations du CV doivent rester strictement fidèles au CV de base.

Il est autorisé de :

- reformuler ;
- raccourcir ;
- réorganiser ;
- mettre en avant ;
- adapter le vocabulaire à l'offre.

Il est interdit de :

- inventer une expérience ;
- inventer un diplôme ;
- inventer une certification ;
- inventer une compétence non présente ;
- modifier une date, un lieu ou un niveau réel.
