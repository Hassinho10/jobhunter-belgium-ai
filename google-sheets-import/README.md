# Import Google Sheets FOREM

Ce dossier contient des CSV prêts à importer manuellement dans Google Sheets.

Nom recommandé du fichier Google Sheets : **Suivi JobHunter Belgium AI - FOREM**.

## Méthode d'import

1. Créer un nouveau fichier Google Sheets.
2. Importer chaque CSV de ce dossier.
3. Créer un onglet par CSV, en conservant l'ordre et les noms proposés.
4. Vérifier les accents et les séparateurs après import.
