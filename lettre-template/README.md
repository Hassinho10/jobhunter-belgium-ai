# Lettre Template

Ce dossier contient la lettre de motivation HTML de base.

## Fichiers

- `index.html` : lettre de motivation HTML utilisée comme modèle principal.
- `Lettre_Motivation_Hassan_Chajai_BASE.html` : copie de référence de la lettre fournie.

## Règle importante

La lettre peut être personnalisée selon une offre, mais elle doit rester fidèle au profil réel.

Il est autorisé de :

- adapter l'objet de la lettre ;
- reformuler les paragraphes ;
- mettre en avant les compétences et motivations pertinentes ;
- raccourcir ou réorganiser le texte.

Il est interdit de :

- inventer une expérience ;
- inventer un diplôme ;
- inventer une compétence ;
- inventer une disponibilité ;
- exagérer le niveau réel du candidat.
