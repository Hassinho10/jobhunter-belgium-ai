# Questions de clarification - TEST-001

Ces questions servent à clarifier les points incertains avant toute génération finale de CV, lettre ou message.

Tant qu'une information reste à confirmer, la génération finale doit rester bloquée.

## Outils numériques

1. Quel est ton niveau réel sur Google Sheets ?
   Options : aucun / débutant / intermédiaire / avancé

2. Quel est ton niveau réel sur Google Forms ?
   Options : aucun / débutant / intermédiaire / avancé

3. As-tu déjà utilisé n8n ?
   Options : non / en cours d’apprentissage / oui débutant / oui intermédiaire

4. As-tu déjà utilisé Make ?
   Options : non / en cours d’apprentissage / oui débutant / oui intermédiaire

5. As-tu déjà utilisé Notion ?
   Options : non / débutant / intermédiaire / avancé

6. As-tu déjà utilisé Trello ?
   Options : non / débutant / intermédiaire / avancé

## Positionnement

7. Es-tu d’accord pour présenter ton profil comme assistant digital junior / automatisation junior ?
   Options : oui / non / à reformuler

8. Es-tu disponible pour un stage ?
   Options : oui / non / à préciser

9. Quelle est ta date de début possible ?
   Réponse libre

10. Quelle durée de stage recherches-tu ?
    Réponse libre

11. Autorises-tu la mention du projet personnel JobHunter Belgium AI dans les candidatures ?
    Options : oui / non / seulement si pertinent

## Rappel

- Ne pas inventer de compétence.
- Une compétence en cours d’apprentissage peut seulement être présentée comme apprentissage.
- Une réponse `non` ne doit pas être utilisée comme argument fort.
- Une information non confirmée bloque la génération finale.
