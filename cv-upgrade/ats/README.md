# CV ATS sobre

Ce dossier contient la version sobre et compatible ATS du CV d'Hassan :

- `cv-hassan-ats.html`
- `cv-hassan-ats.pdf` si la génération PDF locale réussit.

Objectif de cette version :

- fond blanc et texte noir ;
- structure simple sans QR code ;
- mise en page lisible à l'impression ;
- sections claires pour les plateformes de candidature et les ATS ;
- mêmes informations réelles que la version visuelle ;
- projets personnels placés au centre du CV.

À utiliser plutôt pour :

- plateformes de recrutement ;
- candidatures avec formulaire ;
- impression papier ;
- recruteurs ou systèmes qui lisent mal les CV très graphiques.

À compléter avant usage intensif :

- liens GitHub et démos ;
- stacks exactes des projets ;
- dates précises des expériences pharmaceutiques ;
- date de fin exacte de la formation Technofutur TIC.
