# CV visuel modernisé

Ce dossier contient la version visuelle améliorée du CV d'Hassan :

- `cv-hassan-visual-upgraded.html`
- `cv-hassan-visual-upgraded.pdf` si la génération PDF locale réussit.

Objectif de cette version :

- garder l'identité sombre/cyan du CV actuel ;
- rendre la hiérarchie plus claire ;
- mettre les projets personnels au centre ;
- réduire la redondance des liens GitHub, LinkedIn et QR codes ;
- valoriser la reconversion, le parcours scientifique, la rigueur, les GMP et la traçabilité ;
- conserver uniquement des informations réelles ou marquées `à compléter`.

À utiliser plutôt pour :

- envoi direct à une PME, startup, agence web ou recruteur humain ;
- portfolio ou candidature où l'identité visuelle peut aider ;
- présentation locale avant export PDF.

À vérifier avant usage :

- les liens GitHub et démo des projets ;
- la taille finale en PDF ;
- les dates précises Catalent / Novo Nordisk ;
- la date de fin exacte de la formation Technofutur TIC.
